# Anki-deck-Anvendt-statistik
This deck is created for the course _Applied statistics_ for engineering students at Aalborg university (AAU), which is in danish, and the deck is thus primarily in danish.

----

Dette deck er udarbejdet ud fra kurset _Anvendt statistik (ASTA)_ for ingeniørstuderende på Aalborg universitet (AAU) ud fra forelæsningerne og kursets bog, _[WMMY] Walpole, Myers, Myers, Ye (2017) : Probability & Statistics for Engineers and Scientists Global Edition (9th edition)_.
  Da kurset foregår på dansk, er kortene hovedsageligt lavet på dansk med engelske oversættelser for at gøre samlæsning med kursets bog, som er på engelsk, nemmere.

Da jeg i skrivende stund er i gang med kurset, vil decket blive opdateret løbende med tilføjelser til eksisterende kort og nye kort. Når kurset er overstået, vil jeg uploade decket til AnkiWeb.
